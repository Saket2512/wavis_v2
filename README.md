# WAVIS v2 – Bioacoustic Bird Classification

Deep learning pipeline for bird sound classification using:
- BirdCLEF 2021 & 2023
- ESC-50
- EfficientNet + PyTorch

## Features
- Multi-dataset training
- Mel-spectrogram pipeline
- EfficientNet model
- Precomputed mel caching

## Run Training
```bash
pip install -r requirements.txt
python train.py
